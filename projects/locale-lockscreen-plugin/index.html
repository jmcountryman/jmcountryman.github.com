Locale Lockscreen Plugin

This Locale plugin can enable or disable the lock pattern when you turn your screen on.

Disclaimer

This plugin only works on versions of Android older than Froyo. In order to enable and disable the lockscreen this plugin accesses Settings.System, which is where the lock screen preferences are stored in Android 2.1 and below. In Froyo, Google moved these preferences to Settings.Secure, which cannot be accessed by non-system applications.
